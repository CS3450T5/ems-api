# EMS api

## Setup
I'll set up a docker file later, but for now you just need set up a venv with

`python -m venv .venv`

Then activate with

```
Windows:
.venv\Scripts\activate.bat

Linux/Mac:
source .venv/bin/activate
```

Then install the required packages

`pip install -r requirements.txt`
